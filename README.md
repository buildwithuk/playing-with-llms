### Local LLM Interface

A high-performance `Vite` + `TypeScript` + `React` frontend that segregates and streams real-time reasoning tokens and final responses from local Ollama instances. Features a three-column layout locked strictly to the monitor viewport.

*Directory*: `running-llms-locally`

### Openrouter LLM Interface

A lightweight, full-stack client interface built with React, TypeScript, and Bootstrap designed to stream real-time large language model completions. This interface specifically tracks and segregates the internal "thought process" (reasoning tokens) from the final conversational output, creating an optimized layout for deep reasoning models.

*Directory*: `openrouter-reasoning-interface`

## Tech Stack

- **Framework:** React 18+ (Vite SPA template)
- **Language:** TypeScript
- **Styling Framework:** Bootstrap 5 (CSS implementation)
- **API Integration:** Official `@openrouter/sdk`

You will need an OpenRouter API key to use this interface. Sign up at [OpenRouter](https://openrouter.ai/) to obtain your API key.
